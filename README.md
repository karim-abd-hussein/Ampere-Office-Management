A comprehensive Laravel and Filament application designed for companies that manage generators, subscribers, service areas, and field collectors. This system simplifies operations by automating invoice generation, tracking payments, and issuing receipts.

## 🚀 Features

- **Centralized Management**: Manage generators, subscribers, service areas, and field collectors in one platform
- **Automated Invoicing**: Generate and track invoices seamlessly
- **Receipt Generation**: Automatically create and issue payment receipts
- **Area-based Organization**: Assign subscribers to specific service areas
- **Collector Management**: Track field collectors and their assignments
- **Intuitive Admin Panel**: Built with Filament PHP for smooth user experience

## 🛠️ Tech Stack

- **Backend Framework**: Laravel 10+
- **Admin Panel**: Filament PHP
- **Database**: MySQL
- **Frontend**: Tailwind CSS, Livewire
